# Project Purpose and Tech Stack

## Purpose
A modern virtual keyboard for Linux (X11) specially designed for tablets. It aims to be "extremely easy and clear to edit and maintain by an AI agent" by following DRY, SOLID, KISS, and LEAN principles.

## Tech Stack
- **Languages**: C (GCC)
- **Graphics**: Cairo (Vector graphics, antialiasing)
- **Windowing/Events**: X11 (libX11, libXtst for input simulation)
- **Fonts**: FontConfig
- **Build System**: GNU Make
